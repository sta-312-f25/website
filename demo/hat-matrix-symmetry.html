What Symmetry Means:

Hij = Hji means:

  • How much point i influences point j’s fitted value = How much point j influences point i’s fitted value
  • This reciprocal relationship holds for ALL pairs of points
  • Reciprocal influence is a geometric consequence of orthogonal projection
